A configurable, modular, open source charge point

Hardware-agnostic

Runs on any Linux based machine. Supports both off-the-shelf and commercial hardware which can easily be configured to your specific use case.

Cutting edge technology

Written in Go, designed to be efficient and reliable. Supports both OCPP 1.6 JSON/WS and 2.0.1 protocols. Comes with a beautiful admin interface written in Svelte. Also comes with gRPC APIs so you can implement your own applications!

Open source

Best part? It's open source! Modify it, extend it, or use it as a base for your own projects or tailor it to your specific needs.
